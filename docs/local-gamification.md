# Local-First Gamification

Ignight now has a no-server progression layer built for the hot web/PWA edition.

## Included
- Rituals: curated local session arcs selected after game mode.
- Paths: grouped ritual progressions.
- Seals: private achievements shown only in Afterglow and Collection.
- Embers: local progression earned after completed sessions.
- Unlocks: atmosphere milestones opened by Ember totals.
- Chemistry: local tag/mood reads based on cards drawn and outcomes.
- Results screen: compact match stats, embers, unlocks, rituals, seals, paths, and atmospheres.

## Not Included
- Accounts
- Backend
- Multiplayer
- Cloud saves
- Payments
- Remote deck updates

## Storage
Progress is stored locally under `ignight.progress.v1`.
